WHAT THE DOCUMENTATION SPECIFIES

Google Form:
- Name: "Locali MVP Beta — Bug Report"
- 10 required fields (Platform, Device, Version, Type, Steps, Expected, Actual, Screenshot, Name, Email)
- Auto-collects email addresses
- Shows progress bar
- Sends confirmation on submit

Trello Board:
- Name: "Locali — Beta Bugs"
- 5 workflow lists: New → Reviewing → In Progress → Ready for Verification → Done
- 12 color-coded labels (severity + priority)
- 8 Butler automation rules (auto-label, auto-escalate, auto-move)

Zapier Automation:
- Trigger: New Google Form response
- Action: Auto-create Trello card
- Mapping: All form fields → card description
- Label automation: Severity-based labeling
- Test: Submit bug to form, card appears in Trello within 5 minutes

TestFlight & Google Play:
- Minimum 5 testers per platform
- Internal testing groups
- Tester playbook link shared
- Form link shared

Demo Video:
- 2-3 minutes end-to-end
- Shows install → onboarding → home → booking → feedback form
- Shows form submission → Trello card creation
- Narration: "Feedback automatically flows into bug tracking"
